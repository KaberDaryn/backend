## Description
This project demonstrates backend API integration using Node.js and Express.
Weather data is retrieved from OpenWeather API and country data from RestCountries API.

## Selected City
Almaty, Kazakhstan

## APIs Used
- OpenWeather API
- RestCountries API

## Mock Data Mode
Due to OpenWeather API key activation delay, mock data mode is enabled.
The system can switch to real API data by updating environment variables.

## Run Project
npm install  
npm start  

Open in browser:
http://localhost:3000
